# primary-diagonal-sum-c
A simple C program that prints the primary diagonal elements of a 3x3 matrix and calculates their sum.

# Primary Diagonal Sum in Matrix (C Program)

## 📌 Description
This program prints the primary diagonal elements of a 3×3 matrix and calculates their sum.

## 🚀 How It Works
- Traverse the matrix using a loop.
- Access diagonal elements using arr[i][i].
- Add each diagonal element to the sum.
- Print the diagonal elements and their total sum.

## 📥 Input Matrix

```text
1 2 3
4 5 6
7 8 9
```

## 📤 Output

```text
Diagonal Element = 1
Diagonal Element = 5
Diagonal Element = 9

Total Diagonal Sum = 15
```

## 🛠️ Concepts Used
- 2D Arrays
- Matrix Traversal
- Primary Diagonal
- Loops
- Sum Calculation

## 🎯 Learning Outcome
- Understanding matrix diagonals
- Accessing arr[i][i]
- Working with 2D arrays
- Calculating sums in matrices
